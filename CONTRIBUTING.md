# Contributing

Thank you for your interest in **UnitAutogen** — the tSQLt auto-generation framework.

## Current status

This project is at an early stage and is maintained solely by its author,
who is also its sole copyright holder.

**External code contributions are not being accepted at this time.** Pull
requests that add or modify source code or documentation will not be
merged.

This is a deliberate decision. While the project is young, its authorship
and copyright are kept deliberately simple — a single author holding clear
title to the whole work. This may change in the future, at which point a
Contributor License Agreement (CLA) will be introduced and this document
will be updated accordingly.

## What is welcome

- **Bug reports.** Please open an issue describing the problem, the SQL
  Server version you are using, the procedure shape involved, and the
  steps to reproduce.
- **Feature suggestions and feedback.** Please open an issue.
- **Questions.** Please open an issue or a discussion.

When reporting a bug, kindly *describe* the problem rather than pasting a
proposed code fix. The author will implement any change. This keeps the
codebase single-authored and the copyright clean and unencumbered.
